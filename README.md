# forge-core

The Forge bootstrap demo vault. Installed via `[[install]] "forge-core"`
from any Forge-enabled Obsidian vault.

This vault contains a single snippet, `hello_registry`, which returns
the string "hello registry". It exists to verify that the full Forge
stack — backend, plugin, registry, install machinery, cross-vault
snippet resolution, and execution — is working end-to-end.
